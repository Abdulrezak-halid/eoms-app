- This page is used to manage organization partnerships through a token-based connection system.
- Each organization has a unique token that allows other organizations to connect.

- The token is automatically generated for your organization.
  You can share it with another organization to allow them to connect as a partner.
- If the token is regenerated, the previous token becomes invalid.

- To connect to another organization, enter the token they provide.
  The token field is required.
- If the token is valid, the partner will appear in the list.

- The table shows connected organizations and their relation type
