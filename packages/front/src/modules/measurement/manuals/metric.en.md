- This page is used to define and manage metrics that measure and monitor energy performance.

- A metric represents a measurable value such as electricity consumption, temperature, pressure, or flow rate.

- Each metric is linked to a **metric type**, a unit, and optionally an integration source.

### Metric Type

Select how the metric data behaves over time.

- **Counter**  
  Used for values that continuously increase, such as total energy consumption (kWh).  
  The system calculates and displays **delta values**, which represent the difference between consecutive readings.

- **Gauge**  
  Used for values that represent a current state, such as temperature or pressure.  
  The system displays **average, minimum, and maximum** values over the selected time period.

- Choosing the correct metric type ensures accurate calculations and meaningful data visualization.
