- The Data Views page provides a centralized interface for viewing, managing, and analyzing energy-related metrics within the Energy Management System (ENMS).
- It allows users to organize and visualize collected data from different meters, sensors, or systems through configurable metric sets.
- The platform supports real-time and historical data visualization, enabling accurate performance tracking and comparison over time.
- Regularly reviewing and maintaining data views ensures data accuracy, consistency, and meaningful insights for decision-making and reporting.

**Metric Type:**
- Metric: Shows selected metric values directly.
- Meter: Shows selected meter values for a chosen energy resource.
- SEU: Shows selected Significant Energy User values for a chosen energy resource.
